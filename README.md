## How to Run Project
```sh
git clone https://github.com/abiabdillahx/245150701111027-Muhammad-Abi-Abdillah-utptis
cd 245150701111027-Muhammad-Abi-Abdillah-utptis

composer install
cp .env.example .env

php artisan key:generate
php artisan serve
```
Lalu akses http://localhost:8000/
